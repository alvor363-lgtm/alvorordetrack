# نظام تتبع طلبات ورشة النجارة

تطبيق ويب متجاوب لتتبع طلبات ورشة النجارة مع تكامل Firebase.

## المميزات

### للعملاء
- **الصفحة الرئيسية**: واجهة نظيفة مع نموذج تتبع الطلبات
- **صفحة تفاصيل الطلب**: عرض مرئي للتقدم مع الصور
- **متجاوب مع الجوال**: يعمل بشكل مثالي على جميع الأجهزة

### لوحة تحكم المدير
- **تسجيل دخول آمن**: نظام مصادقة بسيط
- **إدارة الطلبات**: إضافة وتعديل وتحديث الطلبات
- **رفع الصور**: رفع صور التقدم لكل مرحلة
- **تحديثات فورية**: تحديث الحالة في الوقت الفعلي

### مراحل الطلب
1. تم استلام الطلب
2. قيد التنفيذ
3. القطع والتجهيز
4. التجميع
5. اللمسات النهائية
6. جاهز للتسليم
7. تم التسليم

## تعليمات التثبيت

### 1. إعداد Firebase
1. أنشئ مشروع Firebase جديد في [Firebase Console](https://console.firebase.google.com/)
2. قم بتفعيل Firestore Database في وضع الاختبار
3. قم بتفعيل Firebase Storage
4. انسخ تكوين Firebase
5. استبدل التكوين النائب في `firebase-config.js`

### 2. تحديث تكوين Firebase
عدّل `firebase-config.js` واستبدل التكوين النائب بتكوين Firebase الفعلي:

```javascript
const firebaseConfig = {
    apiKey: "your-actual-api-key",
    authDomain: "your-project-id.firebaseapp.com",
    projectId: "your-project-id",
    storageBucket: "your-project-id.appspot.com",
    messagingSenderId: "your-sender-id",
    appId: "your-app-id"
};
```

### 3. النشر
- **التطوير المحلي**: افتح `index.html` في متصفح الويب
- **خادم الويب**: استخدم أي خادم ويب (Apache, Nginx, etc.)
- **Firebase Hosting**: انشر باستخدام Firebase CLI

## الاستخدام

### للعملاء
1. قم بزيارة الموقع
2. أدخل رقم طلبك (التنسيق: ORD-001)
3. عرض تقدم الطلب والجدول الزمني

### للمديرين
1. اذهب إلى صفحة المدير
2. سجل الدخول بالبيانات:
   - اسم المستخدم: `admin`
   - كلمة المرور: `admin123`
3. أضف طلبات جديدة أو حدث الطلبات الموجودة
4. ارفع صور التقدم لكل مرحلة

## التخصيص

### الألوان والعلامة التجارية
عدّل متغيرات CSS في `styles.css` لمطابقة ألوان علامتك التجارية.

### مراحل الطلب
عدّل كائن `ORDER_STAGES` في `firebase-config.js` لتخصيص المراحل.

### بيانات اعتماد المدير
حدّث منطق المصادقة في `admin-login.js` للاستخدام الإنتاجي.
